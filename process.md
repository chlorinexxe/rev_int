# Project Setup

This project is a Node.js backend initialized with TypeScript, Express, and SQLite.

## Initial Setup

1. **Create project folders**
   - Created the main project directory
   - Initialized a Git repository
   - Connected the local repository to a remote repo

2. **Backend setup**
   - Created a `backend` folder
   - Initialized npm inside the backend directory:
     ```bash
     npm init -y
     ```

3. **Installed dependencies**
   - Core dependencies:
     ```bash
     npm install express cors better-sqlite3
     ```
   - Development dependencies:
     ```bash
     npm install -D typescript ts-node-dev @types/node @types/express
     ```
